## Blog-it 

---

Blogit is github based blog service 


### Requirements:
    1. Vagrant
    2. Ansible
    
#### Python:
    1. requests
    2. celery
    3. github3.py
    4. Django==1.10.3
    5. djangorestframework==3.5.3
    6. djoser==0.5.4
    7. requests_oauthlib==0.7.0


### Installation:
    1. Run vagrant up
    2. To run localhost, run manage.py server on vm, then open localhost at port 8080
